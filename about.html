Thanks for visiting this site!

C+++ is an open-source C++ IDE on macOS Platform for OI learners to use. It supports Dark Mode and Touch Bar. You needn’t create a Xcode project to compile C++ file or compile a C++ file using the g++ command in the Terminal, all works are done in the C+++ App.

C+++ is Developed by me. I am a student from Shanghai Huayu Private Middle School, class 7 of 2023. I love macOS Developing.
